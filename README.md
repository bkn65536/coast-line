# 資産運用コーストライン シミュレーター

「この年齢で、いくら貯まっていれば**もう積立しなくていいか**」を計算する、Coast FIRE 型のシミュレーターです。

目標金額・目標年齢・想定年率を入力すると、追加の積立をやめても複利だけで目標金額に到達できる「ゴールライン金額」を算出します。

## 特徴

- **入力はシンプル**: 現在の年齢／目標年齢／目標金額／想定年率／現在の資産額（任意）
- **ゴールライン金額を即時計算**: 今いくらあれば積立を止めても目標に届くかを表示
- **年齢別グラフ**: 年齢ごとに必要なゴールライン金額を可視化（自分の現在地もプロット）
- **5年刻みの早見表**: 各年齢で必要な金額を一覧表示
- **依存ライブラリなし**: 単一の `index.html`（HTML / CSS / 素の JavaScript）のみで動作
- **オフライン動作**: ブラウザで開くだけ。データは外部送信されません

## 使い方

`index.html` をブラウザで開くだけです。

GitHub Pages で公開している場合は、公開 URL にアクセスすればそのまま利用できます。

## 計算の考え方

目標年齢までの残り年数を `n`、想定年率を `r` とすると、目標金額 `G` に対して
「今あれば積立しなくてよい金額（ゴールライン金額）」は次式で求めます。

```
ゴールライン金額 = G / (1 + r) ^ n
```

これは、現在の資産を年率 `r` で `n` 年複利運用すると目標金額 `G` に達する元本（現在価値）です。
